# Schelling Segregation Model

This project (first out of two) is realized for the exam 'PHYSICS OF COMPLEX NETWORKS: STRUCTURE AND DYNAMICS'. 
The whole code, the output images and the data produced by the simulation are collected and ordered in the folder shared on this Github. 

The projects analyzed investigate rather complex phenomena that should be analyzed in a much more in-depth than the one I was able to carry out, which are summarized in an exhaustive way in the .pdf file format of the report. Nevertheless, the work I completed significantly exceeded the number of hours originally required by the professor for these projects.

Use of AI tool in this analysis: 
The structure of the code and ~95% of the code is written by me. 
I used AI tools for improve the quality of the plots and to speed up some functions which I coded but  turned out to be slow. Both the original functions and the AI generated ones are showed in the code. Furthermore, the parts written by AI (plots and sweeps) are properly flagged.
The report is polished by AI for a better understanding and a pretty layout.
